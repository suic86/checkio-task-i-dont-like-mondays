You are given a year as an __int__ (e. g. 2016). You should return the number of Monday's in that year.

Input: Year as integer.

Output: Number of Mondays in that year as an int.

Example:

mondays(1911) == 52
mondays(1990) == 52
mondays(1970) == 52
mondays(2015) == 52

Precondition: The year is between 1 and 9999.
